# Trading Bot for Binance Futures Testnet

A command-line trading bot for placing BUY/SELL MARKET/LIMIT orders on Binance Futures Testnet.

## Features

- Connects securely using API keys from environment variables.
- Supports MARKET and LIMIT orders.
- Input validation for all CLI parameters.
- Detailed order summary (ID, status, executed quantity, average price).
- Comprehensive logging of requests, responses, and errors.
- Handles network failures, API errors, and unexpected exceptions.
